# Costs-App

Projeto desenvolvido ao longo do curso de React que eu realizei para iniciar meus estudos nessa biblioteca tão utilizada hoje em dia.

## Tecnologias utilizadas:

 - React
 - Javascript
 - CSS

 ## Como rodar:

- Clonar repositório:

      git clone git@github.com:devBrait/React_NewsLetter.git

- Instalar dependências necessárias:

      npm install

- Rodar projeto:

      npm run dev
